# Usage
<!-- usage -->
```sh-session
$ npm install -g vidiff
$ vidiff COMMAND
running command...
$ vidiff (-v|--version|version)
vidiff/0.0.0 darwin-x64 node-v10.15.3
$ vidiff --help [COMMAND]
USAGE
  $ vidiff COMMAND
...
```
